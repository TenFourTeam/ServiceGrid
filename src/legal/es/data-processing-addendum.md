# Anexo de Procesamiento de Datos

## 1. Definiciones

Estableciendo términos clave.

- Datos Personales: Nombre, email, detalles bancarios procesados vía ServiceGrid.
- Controlador: Tú, determinando el uso de datos.
- Procesador: ServiceGrid, procesando en tu nombre.

## 2. Alcance del Procesamiento

Describiendo las actividades de datos.

- Propósito: Habilitar programación, facturación y servicios de leads.
- Duración: Mientras uses ServiceGrid o según lo requiera la ley.
- Tipos: Datos personales y de pago.

## 3. Obligaciones del Procesador

Detallando las responsabilidades de ServiceGrid.

- Procesar datos solo según tus instrucciones.
- Implementar encriptación y medidas de seguridad.
- Notificar violaciones de datos dentro de 72 horas.

## 4. Subprocesadores

Gestionando la participación de terceros.

- Stripe para procesamiento de pagos.
- Proveedores de nube bajo NDA.
- Lista actualizada en ServiceGrid.com/subprocessors.

## 5. Derechos del Sujeto de Datos

Apoyando el control del usuario.

- Asistir con solicitudes de acceso, corrección o eliminación.
- Contactar privacy@ServiceGrid.com para soporte.
